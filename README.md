# Hierarchical Skill Refinement and Pruning for LfD

This repository contains the implementation of a hierarchical two-phase framework designed to learn "Better-than-Demonstrator" policies from structurally noisy demonstrations. Unlike existing methods like D-REX, which primarily handle fine-grained local noise, this framework explicitly identifies and prunes **structural suboptimality**—entire segments of unnecessary actions (e.g., spinning in place or wandering) that do not contribute to task completion.

## Author
* **Jaemin Cho** – Purdue University, Autonomous and Connected Systems (ACS)

## Problem Statement
Real-world human demonstrations are rarely optimal and typically contain two distinct types of noise:
1. **Local Suboptimality:** Fine-grained inefficiencies within a trajectory (e.g., trembling, suboptimal continuous control).
2. **Structural Suboptimality:** Redundant, task-irrelevant behaviors forming entire segments of the demonstration (e.g., a robot spinning 360 degrees before placing an object).

The objective of this project is to create a learning pipeline that not only smooths out local inefficiencies but also globally optimizes the sequence of skills to strictly eliminate structural noise, yielding a policy that heavily outperforms the human demonstrator.

## Methodology: Two-Phase Framework
This project proposes a hierarchical approach to decompose, refine, and recompose the demonstration data.

### Phase 1: Subgoal Segmentation & Trajectory Refinement
* **Segmentation:** The demonstration is partitioned into subgoals defined by points where task-relevant state changes exceed a specified threshold.
* **Cost-Aware Refinement:** Each segmented trajectory is locally smoothed by minimizing the action cost while maximizing geometric similarity to the original demonstration.

### Phase 2: Skill Learning & Composition Optimization
* **Skill Policy Extraction:** D-REX (Deep Reward Extrapolated) and SSRR algorithms are applied to the refined subgoals to learn distinct skill policies.
* **Skill Composition:** The overall trajectory is formed by combining these skills. A global objective function evaluates the composition by weighing goal achievement, shape similarity, and action cost. 
* **Pruning:** A greedy backward elimination algorithm systematically removes skill segments that increase action cost without aiding goal completion.

## Experimental Setup
The framework is validated using the **MuJoCo Fetch Robotics** environment (e.g., Pick and Place tasks).

### Key Specifications:
* **Environment:** OpenAI Gym `FetchPickAndPlace-v1`
* **Noise Injection:** Demonstrations are manually injected with structural noise (e.g., circular spinning motions).
* **Metrics:** Evaluated against baseline D-REX using cumulative action cost, episode length, and overall task success rate.
